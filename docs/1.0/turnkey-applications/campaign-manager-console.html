Campaign Manager Console

Overview

Telestax Campaign Manager allows Communications Providers to deliver SMS and Voice campaign management solutions to their Business Customers. Manage credits and users with different roles and capabilities, while enabling Business Customers to send the same SMS or make Voice calls to a list of phone numbers. For more information, please check out the Campaign Manager General Documentation.

This document contains information for Communications Providers regarding the Campaign Manager UI Console.

To learn more about the API, have a look at the Getting Started with the Campaign Manager API Guide

Accessing the Campaign Manager Console

You can access the Campaign Manager Console by navigating to https://yourdomain.com/campaignmanagerconsole/.

Business Customer

Communications Providers can create, manage, or delete Business Customer subaccounts through their own Restcomm account. By setting up customer subaccounts, Communications Provider can easily manage billing and invoices for a specific Business Customer.

Business Customers List

Users can view a list of all Business Customers that have been added under their organization. Business Customers can be filtered based on name.

Add Business Customer

Follow the steps below to add a Business Customer:

  • Click on the Add Enterprise button in the Enterprise list page.

  • Enter the name of the new Business Customer.

  • Enter a user name for the account. A user with this name will be created for this account. If nothing is entered, the default value will be the Admin Email Address which is mentioned in the next step.

  • Choose an administrator for this Business Customer. Enter the Admin Email Address.

  • Enter the password for the Admin. This will be set in the Restcomm core platform.

  • Enter the throttle value as explained below

    • smsPerSecond : represents SMS/sec. 0 is used to represent the maximum SMS/sec at a given point in time.

    • callsPerSecond : represents calls/sec. 0 is used to represent the maximum calls/sec at a given point in time.

    • concurrentCalls: represents the number of concurrent calls allowed at any given point in time. 0 is used to represent unlimited amount.

  • Once you are finished, click Save.

Whenever a Communications Provider adds a new Sub Account for a Business Customer, a mandatory User must be defined. This User gets created under the Restcomm Core platform as a Sub Account of the Communications Provider Account.
The Enterprise name and Email ID must be unique.

Passwords should meet the following criteria:

  • At least one Uppercase letter (A,B,C,D…​)

  • At least one Lowercase letter (a,b,c,d,…​…​.)

  • At least one number (1,2,3,4,5.)

  • At least one special character (!,@,#,$,%,&,…​…​.)

  • Length of the password should be greater than or equal to 8.

Edit Business Customer

To edit the details of a Business Customer, click on the Business Customer name from the list. The following parameters can be changed:

  • Name of the Business Customer

  • Throttle value

  • Status

Click save after making the appropriate changes.

Delete Business Customer

To delete a Business Customer click on the customer’s name from the list of Business Customers. Then click on the Delete button on the bottom of the screen. This action cannot be undone. Once the account is deleted you will not be able to restore it.

When a Business Customer is deleted, all the users of that Enterprise are CLOSED in Restcomm.

User Management

To navigate to this page click on User Management in the top menu. Once in the User Management page, users can:

  • View a list of available users along with their roles, date created, and status

  • Filter a specific user based on name

Add User

Follow the steps below to add a user:

  • Click on the Add User button in the User Management page

  • Enter a name for the new user

  • Enter an email address for this user

  • Enter a password for the user

  • Select a role for the user. The possible roles are CP_Administrator and CP_Provisioning_Agent

The user’s Email ID must be unique. If the role of the User is CP_Administrator, you can add smsPerSecond, callsPerSecond and concurrentCalls throttling values for them. Those values should match with the Restcomm platform throttling.

Passwords should meet the following criteria:

  • At least one Uppercase letter (A,B,C,D…​)

  • At least one Lowercase letter (a,b,c,d,…​…​.)

  • At least one number (1,2,3,4,5.)

  • At least one special character (!,@,#,$,%,&,…​…​.)

  • Length of the password should be greater than or equal to 8.

Add new user for Enterprise

To add a new user for an Enterprise, go to the Business Customer List tab by clicking on the Enterprise tab as mentioned in Business Customers List. Click on Users to list all the Users for that Enterprise. Click on Add User button to add new Enterprise User

Edit User

To edit the details of a user, click on the user name from the list of users under the User Management section. The following parameters can be updated:

  • Name: Name of the user

  • Role: Role of the user can be changed only between CP_ADMINISTRATOR (Communication Provider Admin) and CP_PROVISIONING_AGENT (Sub-Account Provisioning Agent)

  • Status: Possible values that can be selected are Active or Inactive. User status can be changed between Active and Inactive

Delete User

An active or an inactive user can be deleted. Once deleted the Status of the account will be set to Closed. The status cannot be changed back to active or inactive. To delete a user click on the name of the user then click the Delete button present on the bottom of the user’s detail page. Once deleted the status of the user will be set to CLOSED.

Credit

Credits are needed to create a campaign successfully. Communications Providers can add or remove credits from the Sub Accounts of their Business Customers.

  • Whenever Campaign is scheduled, the equivalent Credit is deducted.

  • If there is not enough Credit available, scheduling of the Campaign will fail.

  • If a Scheduled Campaign is cancelled, the equivalent Credit is added back.

  • Once a Campaign progresses to InProgress state, Credit cannot be reversed, no matter what.

  • One credit is equal to one SMS or one Voice call.

  • For example, if a particular Business Customer has 100 credits, that customer can only send 100 SMS Segments or make a 100 Voice calls.

Manage Credits

To Manage the Credits of a particular Business Customer do the following:

  1. Click on the name of the Business Customer from the list of Business Customers

  2. Click on the Manage Credits button on the top right corner.

  3. Select Add Credits or Reduce Credits in the Modify Credits Type field

  4. Specify the amount to credit or debit in the Add Credits field

  5. Enter the description for this particular transaction

  6. Click on the Save button

Search

To search Business Customers, Users, Campaigns, and Credits, click on Search in the top menu to navigate to the page. Then click the appropriate option to search from the side menu.

Search Business Customer

A Communications Provider can search Business Customers based on the following filter parameters:

  • Name: Display Business Customers whose name matches partially or fully the entered name.

  • Status: Display Business Customers whose Status is the same as the selected Status.

  • Start Date: Display Business Customers created on or after this date.

  • End Date: Display Business Customers created on or before this date.

Business Customers Details

To view the details of a user, click on the name of that user.

Search User

Search users based on the following filter parameters:

  • User Name: Display Users with Name matching partially or fully with the entered Name.

  • Role: Display Users with Role as the selected Role.

  • Email: Display User with the entered Email ID.

  • Business Customer: Display Users belonging to the selected Business Customer.

  • Start Date: Display Users created on or after this date.

  • End Date: Display Users created on or before this date.

User Details

To view the details of a User click on the name of the particular user.

Search Campaign

Search Campaigns based on the following filter parameters:

  • Name: Display Campaigns with Name matching partially or fully the entered name.

  • Status: Display Campaigns with Status as the selected status.

  • Business Customer: Display Campaigns belonging to the selected Business Customer.

  • Start Date: Display Campaigns created on or after this date.

  • End Date: Display Campaigns created on or before this date.

  • Channel: Display Campaigns with channel matching the selected value.

Campaign Details

To view the details of a campaign, click on the name of the particular user.

Search Credits

Search Credits based on the following filter parameters:

  • Business Customer: Display Credits for the Selected Business Customer.

  • Type: Display Credits with the specified Type. Type can be Credit, Debit or Both.

  • Start Date: Display Credits, credited or debited on or after this date.

  • End Date: Display Credits, credited or debited on or before this date.

Credit Details

To view the details of a Credit click on the name of the user.
