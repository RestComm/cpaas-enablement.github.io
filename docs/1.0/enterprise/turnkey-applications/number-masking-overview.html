Number Masking Overview

Introduction

Number Masking enables Enterprises to easily make communication between two or more parties without revealing private information (such as phone number). Simplicity of the API design allows Enterprises to add cCalled and calling parties in a session without worrying about how masked numbers will be assigned.

Advantages of Number Masking

  • Manage sessions and gather insights: API and UI provides in depth knowledge on conversations happening and learn from interactions.

  • Time to market: Ready turnkey applications allow you to develop apps in a matter of minutes vs leveraging Voice/SMS API and scribbling to manage conversations.

  • Intuitive Console: Manage Applications, Phone Numbers, Mask Pool, CDR’s etc via interactive and intuitive Console interface.
