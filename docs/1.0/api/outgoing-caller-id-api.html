Outgoing Caller ID

OutgoingCallerId

An OutgoingCallerId instance resource represents either a single or a set (through regex) of verified numbers that may be used as a caller ID when making outgoing calls or sending SMS via the REST API and within the RCML <Dial> verb.

The OutgoingCallerIds list resource represents the set of an account’s verified phone numbers.

OutboundCallerId Resource URI.

/2012-04-24/Accounts/{AccountSid}/OutgoingCallerIds/{OutgoingCallerIdSid}

Property

Description

Sid

A 34 character string that uniquely identifies this resource

DateCreated

The date that this resource was created, given in RFC 2822 format.

DateUpdated

The date that this resource last updated, given in RFC 2822 format.

FriendlyName

A human readable descriptive text for this resource, up to 64 characters long. By default, the FriendlyName is a nicely formatted version of the phone number.

AccountSid

The unique ID of the Account responsible for this Caller Id.

PhoneNumber

A valid regular expression. In case of concrete number, it must be formatted with a '+' and country code e.g., ^\+16175551212 (E.164 format).

Uri

The URI for this resource

Supported Operations

HTTP GET

Gets the details of a particular Outgoing Caller ID.

curl -X GET 'https://{organization}.restcomm.com/restcomm/2012-04-24/Accounts/{AccountSid}/OutgoingCallerIds/{IncomingPhoneNumberSid}.json' -u {AccountSid}:{AccountToken}

HTTP PUT

Updates the caller ID, and returns the updated resource if successful.

Only CSP accounts can update existing records.

Property

Description

FriendlyName

A human readable descriptive text for this resource, up to 64 characters long. By default, the FriendlyName is a nicely formatted version of the phone number.

PhoneNumber

A valid regular expression. Single numbers shall be formatted with a '+' and country code e.g., ^\+16175551212 (E.164 format).

 
curl -X PUT https://{Organization}.restcomm.com/restcomm/2012-04-24/Accounts/{AccountSid}/OutgoingCallerIds/{PhoneNumberSid}.json \
--data-urlencode "FriendlyName=John Doe" \
-u {AccountSid}:{AccountToken}

}
HTTP DELETE

Deletes the caller ID from the account. Returns an HTTP 204 response if successful, with no body.

Only CSP accounts can delete existing records.

curl -X DELETE 'https://{organization}.restcomm.com/restcomm/2012-04-24/Accounts/{AccountSid}/OutgoingCallerIds/{IncomingPhoneNumberSid}.json' -u {AccountSid}:{AccountToken}

OutgoingCallerIds List Resource URI

/2012-04-24/Accounts/{AccountSid}/OutgoingCallerIds

Supported Operations

HTTP GET

Returns a list of OutgoingCallerId resource representations, each representing a Caller ID number valid for an account. The list includes paging information.

HTTP POST

Adds a new Caller ID to an Account.

Only CSP accounts can create new records.

The following parameters are accepted:

Parameter

Description

PhoneNumber

A valid regular expression. For specific phone numbers, it should be formatted with a '+' and country code e.g., ^\+16175551212 (E.164 format).

FriendlyName (optional)

A human readable description for the new caller ID with maximum length 64 characters. Defaults to a nicely formatted version of the number.

 
curl -X POST https://{Organization}.restcomm.com/restcomm/2012-04-24/Accounts/{AccountSid}/OutgoingCallerIds.json \
--data-urlencode "FriendlyName=My Home Phone Number" \
--data-urlencode "PhoneNumber=^\+14158675310" \
-u {AccountSid}:{AccountToken}
