Restcomm Reason Codes Dictionary

Restcomm will provide a reason code along with a short description for failed or blocked session attempts, such as voice or sms sessions.

Reason code and the relevant description will be provided in either SIP or HTTP responses.

Reason codes 0 - 999 are reserved for network specific error codes from network protocols such as SIP, HTTP, MGCP etc.
