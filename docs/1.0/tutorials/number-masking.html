Phone Number Masking

In this tutorial, you will learn how to create a number masking application for a taxi service with Telestax Number Masking. The application will connect the customer and the taxi driver through masked phone numbers, while their identity stays protected. Follow the steps below to get started.

Prerequisites

In order to be able to set up the application you will need to have an Admin account under your Communications Provider Restcomm organization.

Creating an Enterprise Account

To start with, you will need to set up an enterprise account.

  • Login in to your Communications Provider account

  • Navigate to the Number Masking Console https://gss.restcomm.com/numbermaskingconsole/enterprise/list

  • Click on the Add Enterprise button

  • Fill out the Create Enterprise form with the necessary details

  • Click on the Done button

This will create a new enterprise and will also add a new user account that the enterprise must use to log into the Number Masking Console.

If the enterprise is successfully created, you will be able to see it in the list of existing enterprises under your account.

Registering the Masked Numbers

Next, you will have to register two phone numbers that will later be used as masked numbers in order to protect the identity of the customer and the taxi driver.

  • Log into the Restcomm Dashboard using the account created in the previous step

  • Navigate to the tab Numbers → Register Number → Provider Number

  • Select the country and area codes

  • Set the Capabilities to Voice

  • Register two numbers as shown below

You need a minimum of two registered phone numbers in order for Number Masking to work.

Creating Your Number Masking Application

  • Navigate back to the Number Making Console

  • Click on the tab Numbers

  • The phone numbers registered in the previous step will be available in the list of numbers, similar to the screenshot below.

  • Click on the Applications tab

  • Click on the Add Application button

  • Add the name of the application and press Done.

The newly created application will appear in the list of Applications.

Adding Masked Numbers to Your Application

  • Click on the name of the newly created application

  • Go to Mask Numbers → Add Mask Number

  • Select from the list and add the two numbers

Adding Context to Your Application

  • On the same screen, navigate to Context → Add Context

  • Fill the Context form as shown below in order to add a new Context

  • Click Done

Adding Participants to the Context

  • Click on the Participants link in the newly created context

  • Click on the Add Participants button

  • You will need to add two participants (Customer and Taxi driver) to the Context

  • Fill out the Add Participants form with the customer details

    NOTE: The phone number field corresponds to the customer’s actual phone number. The Mask number field is the customer’s masked number registered in the previous steps.

  • Click Done

Repeat the steps above on the behalf of the taxi driver

Your Number Masking Application in Action

You are all set! Every time the customer calls a taxi from his number +1111 to +13053902732 his identity will be protected. The taxi driver will receive the call and will see +13053878126l as a caller ID.
